# EasyClient #

EasyClient是EasyDarwin开源流媒体云平台的客户端实现，包含：Windows、Android、iOS三个部分，其主要功能包括：

1. 云平台设备列表获取；
2. 设备实时码流请求与播放；
3. 设备云台控制；
4. 设备语音对讲；


## 更新记录 ##

### EasyClient(Windows) v1.0.16.08.15 ###

* EasyClient(Windows) 作为EasyCMS云平台的客户端，实现获取设备列表，获取设备流URI进行直播,摄像机PTZ云台控制等功能；
